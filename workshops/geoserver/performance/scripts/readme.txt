The files in this folder are:

* wms_request.py     - A script that generates random bounding boxes and outputs them to csv files. This is so we rerun the same test with the same random bounding boxes used to control the WMS request.
* bbox-commands.txt  - A text file that contains the commands used to get wms_request.py to generate random bounding boxes for the csv files
* roads-bbox.csv     - Random bounding boxes limited to extent of roads dataset
* raster-bbox.csv    - Random bounding boxes for whole world tests used for raster test
* vector-bbox.csv    - Random bounding boxes for whole world tests used for vector test
